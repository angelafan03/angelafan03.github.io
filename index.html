GDP and Life Expectancy Around the Globe

Project created during spring quarter 2022 for the class Python with Applications I. Using data from Gapminder, I wanted to visualize and compare GDP per capita and life expectancy of continents across time, as well as the effect of GDP on life expectancy. .

Data Compared: GDP per capita and life expectancy.

Python Packages used: NumPy, Pandas, Matplotlib, Seaborn

Data visualization using Python
